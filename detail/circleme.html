CircleMe

Private Social Community Application

Flutter Firebase Social Media Mobile App Community Platform

Project Overview

CircleMe adalah aplikasi mini social media berbasis Flutter yang dirancang untuk komunitas kecil agar dapat berbagi postingan, berinteraksi, dan mengelola circle secara privat dan terstruktur.

Main Features

  • Create & Join Private Circle
  • Timeline / Feed per Circle
  • Comment & Chat System
  • Photo & Media Album
  • Event & Notification System

Technology Stack

  • Flutter (Frontend)
  • Firebase Authentication
  • Firestore Database
  • Firebase Storage

Target Users

  • Keluarga
  • Kelas Sekolah / Kampus
  • Komunitas Hobi
  • Organisasi Kecil
