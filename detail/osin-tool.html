ODDNEY

Advanced Open Source Intelligence & Digital Footprint Analysis Tool

Python OSINT Automation Digital Forensics Cyber Security

Project Overview

ODDNEY adalah tool OSINT berbasis Python yang dirancang untuk melakukan pencarian informasi publik secara otomatis dari berbagai sumber online seperti Google, Twitter, LinkedIn, Instagram, dan email discovery.

Objectives

  • Mengotomatisasi proses Open Source Intelligence
  • Menganalisis digital footprint target
  • Mengintegrasikan berbagai sumber pencarian
  • Menyederhanakan investigasi berbasis OSINT

Key Features

  • Google Search Automation
  • Social Media Lookup
  • Email Finder Integration
  • CLI-based Interface
  • Lightweight & Fast Execution

System Workflow

  • User inputs target name
  • Tool generates search queries
  • Collects public data links
  • Outputs structured intelligence results
