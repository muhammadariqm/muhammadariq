IoT Smart Lamp

Voice Controlled Smart Home Lighting System

Arduino IoT Voice Recognition Smart Home Embedded System

Project Overview

Sistem Smart Lamp berbasis IoT yang memungkinkan pengguna mengontrol lampu menggunakan perintah suara melalui modul mikrokontroler dan integrasi sensor.

Objectives

  • Mengimplementasikan kontrol suara
  • Mengembangkan sistem smart home sederhana
  • Mengintegrasikan hardware & software
  • Meningkatkan efisiensi penggunaan listrik

Hardware Components

  • Arduino Board
  • Relay Module
  • Voice Recognition Module
  • LED / Lamp Module
  • Breadboard & Jumper Cables

System Workflow

  • User gives voice command
  • Voice module processes command
  • Arduino validates instruction
  • Relay triggers lamp ON/OFF
